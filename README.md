# learning — mon parcours de programmation

Ce dépôt est mon carnet d'apprentissage de la programmation, encadré par un
professeur particulier (le skill **prof-dev**). J'y apprends **Python, SQL,
JavaScript et TypeScript** en partant de la programmation à zéro, avec une
orientation **automatisation système & réseau** (profil SISR).

## Comment ça marche

- **Le prof ne code jamais à ma place.** J'écris 100 % du code et je tape
  moi-même toutes les commandes. C'est comme ça qu'on apprend vraiment.
- Chaque séance suit un déroulé : révision éclair → un concept → un exercice →
  relecture de mon code → mise à jour de la mémoire du prof.
- ~20 % théorie, ~80 % pratique.

## Structure du dépôt

```
learning/
├── .claude/skills/prof-dev/   Le skill du professeur (SKILL.md)
├── .mentor/                   La mémoire du professeur
│   ├── PROGRESSION.md         Où j'en suis (concepts acquis / fragiles / à voir)
│   ├── CURRICULUM.md          Le plan de formation complet
│   ├── revisions.md           File de révision espacée
│   └── journal/               Compte-rendu de chaque séance
├── python/                    Mes exercices Python
├── sql/                       Mes exercices SQL
├── javascript/                Mes exercices JavaScript
└── typescript/                Mes exercices TypeScript
```

## Reprendre une séance

Ouvrir le projet et dire simplement : **« Session de cours, on continue. »**
Le prof relit sa mémoire (`.mentor/`) et enchaîne là où on s'était arrêtés.
