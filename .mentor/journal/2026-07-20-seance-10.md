- **`continue`** = sauter directement au tour suivant.
- Bonne question spontanée de l'élève : « peut-on mettre plusieurs `except` après un `try` ? » → oui (un par type d'erreur, ou regroupés). Curiosité et anticipation = bon signe.

## Ce qui a été construit — `python\subnet_projet.py` (version finale)
Boucle `while True` : `try` lit le préfixe en int ; `except ValueError` → message + `continue` ; sinon `if 0<=prefixe<=32` → affiche hôtes + `break`, `else` → message hors intervalle. Puis classification `if`/`elif`/`else`.
- Critères OK : `abc` (pas de crash), `99` (message hors 0–32), `24` (→ 254, Grand reseau).
